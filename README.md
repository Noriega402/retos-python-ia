# Retos de Python - IA

Este repositorio contiene una serie de ejercicios básicos de Python realizados en Google Colab como parte de práctica y aprendizaje en programación.

## Contenido

Los ejercicios incluyen:

* Listas (meses del año y acceso por índice)
* Creación de objetos (diccionarios y clases)
* Estructuras condicionales (`if`, `elif`, `else`)
* Operadores lógicos
* Bucles (`for`, `while`)
* Funciones
